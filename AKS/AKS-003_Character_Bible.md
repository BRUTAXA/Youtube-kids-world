# AKS-003 — Character Bible

Status: Draft
Version: 0.1

## Purpose

AKS-003 defines the main ensemble of AI Kids Studio / Luma.

The goal is to make the characters simple enough for children aged 2–4 to recognize, but consistent enough for writers, AI systems, artists and future automation.

## Core principle

For ages 2–4, each character must be instantly recognizable.

Each character needs:

- one main trait;
- one visual color;
- one recognizable external feature;
- one question area;
- one strength;
- one difficulty;
- one short repeating phrase.

A child should be able to understand the character quickly, even before remembering the name.

## Rules for ensemble stories

- Do not put all six children in focus at once.
- One episode focuses on one or two children.
- Other characters can appear in the background or as light support.
- Each child should get fair screen time across a season.
- Weaknesses must have a sympathetic side.
- Avoid gender stereotypes.
- Avoid sarcasm and adult-style irony.
- Keep characters emotionally simple but warm.
