# AI Kids Studio Bible

The Bible contains assets and rules that survive across episodes.

It is the home for long-term studio knowledge:

- foundation principles;
- visual style;
- world rules;
- approved characters;
- reusable design references.

## Rule

The Bible contains what is reusable across many episodes.

Episode-specific work belongs in `episodes/`.

Decisions belong in `decisions/`.

Observations from children belong in `observations/`.
