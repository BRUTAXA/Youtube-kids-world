# AI Kids Studio

Добро пожаловать в официальный репозиторий проекта **AI Kids Studio**.

## Миссия
Создать лучший в мире образовательный AI-проект для детей, основанный на психологии развития, настоящих детских вопросах, добрых историях и проверке решений на реальных детях.

## Текущий статус

**Release:** `v0.1.0 — Foundation Complete`  
**Phase 1:** ✅ Foundation Complete  
**Current Phase:** 🎨 Phase 2 — Visual Research  
**Next Milestone:** `ART-001.001 — Eye Research`

## Главный принцип

> **Reality over Architecture.**

Архитектура проекта считается достаточной. Дальше качество проекта определяется не количеством новых документов, а качеством изображений, аниматиков и реакцией детей.

## Структура проекта

```text
FOUNDATION
├── AKS/      — философия, психология, мир, персонажи, визуальный язык

RESEARCH
├── ART/      — визуальные исследования и эксперименты
├── DEC/      — design decisions: почему принято решение
└── OBS/      — наблюдения за реакцией детей и родителей

PRODUCTION
└── STORY/    — сценарии и reference stories
```
